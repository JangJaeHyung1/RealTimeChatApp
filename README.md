# MessageKit을 사용한 실시간 Chat 앱

## 적용 사항
- Firebase Auth와 GoogleSignIn로 구현한 구글 로그인, 이메일 로그인
- 실시간 대화 기능
- location 메세지 기능
- photo, video 메세지 기능
- modal에서 이름 키워드로 사용자 검색, 검색된 사용자와 메세지 주고 받기
- 대화목록에서 대화 삭제
- 다크모드 적용

### 그 외
- SDWebImage을 사용하여 이미지 set 캐시를 줄였으며
- MapKit과 CoreLocation을 사용하여 위치 정보와 지도에 위치를 나타내도록 함
- MVC 패턴으로 디자인 구성
